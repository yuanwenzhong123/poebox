# 双瓦腐化利润计算器 V0.1

## 📌 产品概述

**双瓦腐化利润计算器**是一款专为《流放之路》（Path of Exile）玩家设计的本地化辅助工具，帮助玩家**系统追踪、记录和分析双瓦腐化装备的成本与收益**，告别凭感觉做生意，用数据驱动交易决策。

---

## 🧩 核心功能模块

### 1. 多装备管理
- 支持同时管理**多件装备**的腐化记录，通过左侧侧边栏快速切换
- 每件装备独立保存名称、成本配置和腐化历史，互不干扰
- 支持装备的新建、删除和搜索

### 2. 成本配置
- 可分别设置**装备本体价格**和**瓦尔宝珠费用**
- 价格单位支持 **神圣石（D）** 和 **混沌石（C）** 自由切换
- 可设置**神圣/混沌兑换比率**，自动换算统一核算

### 3. 腐化记录表（核心功能）
逐条记录每次腐化结果，每条记录支持四种结果标注：

| 结果 | 说明 |
|---|---|
| 🟢 腐化成功 | 获得目标词缀 |
| 🔴 装备消失 | 装备被销毁 |
| 🟡 已腐化无变化 | 变成已腐化状态，无新词缀 |
| ⬜ 全部变白孔 | 词条变白但有孔位 |

每条记录可填写：卖出价格、成本、兑换汇率，表格自动计算**单次利润/亏损**。

### 4. 数据统计面板
实时汇总当前装备的所有腐化数据：
- 总腐化次数、各结果次数与占比
- 总成本、总收入、**总利润**
- 可视化进度条展示各类结果比例分布

### 5. 腐化词缀配置
- 内置可编辑的腐化词缀列表，涵盖生命、元素、法力、防御、宝石等各类词缀
- 每个词缀支持设置**权重**，影响模拟器的概率分布
- 支持添加/删除自定义词缀

### 6. 模拟器（双模式）
**自动模拟模式：**
- 设置模拟次数（如100次）一键批量模拟，快速预估期望收益
- 自动汇总成功率、消失率及预估利润

**手动单次模拟模式：**
- 每次点击手动触发一次腐化，即时反馈结果
- 成功时显示随机抽取的两条词缀及其权重
- 每次结果附带风趣的**随机评语**（差运版/欧皇版/普通版），极具游戏氛围感

### 7. 知识库
内置双瓦腐化机制说明，包括各结果的理论概率（各约25%），以及宝石双瓦的特殊计算器，帮助新手快速了解腐化机制。

### 8. 数据导入 / 导出
- 支持将所有装备数据**导出为 JSON 文件**本地备份
- 支持从 JSON 文件**导入恢复数据**，方便跨设备使用或分享给队友

### 9. 笔记功能
附带独立笔记模块，可记录交易心得、市场行情、进货思路等自由文本，支持搜索。
